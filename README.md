# Portfolio — Dounia Huber

## Description

Ce projet est mon portfolio personnel, qui clôture ma formation d’intégratrice web chez OpenClassrooms. Il présente mon parcours, mes projets concrets et mes compétences techniques en HTML, CSS, JavaScript et React, tout en respectant les bonnes pratiques en matière de SEO, accessibilité et performance.

## Technologies

- React
- JavaScript
- Tailwind CSS
- Vite
- EmailJS

## Installation

```
git clone https://github.com/Dounia44/Portfolio
cd portfolio
npm install
npm run dev
```
